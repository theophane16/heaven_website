Listes d'outils :

  • Vert → Utilisé, maitrisé (au moins en partie)
  • Orange → ceux sur la liste des apprentissage imminents.
  • Bleu → Connaissances (de nom, utilité) mais peu / pas utilisé pour le moment, liste des apprentissages futurs.
  • Gris → Plus des techniques que des outils.
  • En gras → Que je considère comme centraux.
  • Google Dorks
    → Recherche optimisée : une technique qui utilise des opérateurs avancés dans Google (ex: site:, filetype:, inurl:) pour trouver des informations sensibles ou indexées par erreur sur le web.
  • Google Lens
    → Recherche inversée : un outil de reconnaissance d'images par Google qui permet d'identifier des objets, du texte, des lieux et de rechercher des informations à partir d'une photo.
  • Shodan
    → Moteur de recherche pour trouver des appareils connectés à Internet (caméras, serveurs ...)
  • Maltego
    → Outil d'analyse graphique pour l'OSINT et la cartographie des relations en cybersécurité.
  • Réseaux sociaux
    → Facebook / LinkedIn → Sources majeures d'OSINT pour l'ingénierie sociale.
  • Overpass turbo
    → Cartographie très complète (chercher tout et n'importe quoi)
  • Sherlock / Maigret
    → Recherche de profils d'utilisateurs sur plusieurs plateformes de réseaux sociaux.
  • Dehashed
    → Moteur de recherche de bases de données contenant des informations compromises (emails, mots de passe...)
  • Epieos
    → Recherche d'informations à partir d'une adresse email ou d'un numéro de téléphone.
  • Exiftool
    → extraire des métadonnées de fichier
  • Waymore
    → Outil d'OSINT permettant de récupérer des sous-domaines à partir de Wayback Machine.
  • The Harvester
    → Collecte des emails, noms de domaines et informations liées à une entreprise.
  • Bypass UrlPasser
    → Contourne les protections de certains raccourcisseurs d'URL.
  • Firefly
    →Outil de phishing avancé permettant de générer de faux sites de connexion.
  • HexHTTP
    → Outil d'analyse des requêtes HTTP.
  • 404notfound
    → Outil d'OSINT permettant de récupérer des pages supprimées.
  • Maltego
    → Outil d'analyse graphique pour l'OSINT et la cartographie des relations en cybersécurité.
  • Google / Google Dork
    → basique mais très utile pour trouver des informations précises parmi touts les sites existants.
  • Metasploit
    → Framework d'exploitation de vulnérabilités pour tests d'intrusion.
  • Social Engineering Toolkit (SET)
    → Suite d'outils pour tester des attaques d'ingénierie sociale.
  • Shoulder surfing
    → Technique d'attaque basée sur l'observation visuelle des mots de passe.
  • Gophish
    → Plateforme de test de phishing pour entraîner les utilisateurs.
  • SocialFish
    → Framework pour la création de fausses pages de connexion (phishing).
  • CredSniper
    → Capture de mots de passe via du phishing avancé.
  • ReelPhish
    → Outil automatisé de phishing qui intercepte des sessions en direct.
  • Cyberchef
    → Outil polyvalent pour transformer, encoder et décoder des données
  • Metasploit
    → Framework d'exploitation de vulnérabilités pour tests d'intrusion.
  • Burp suite / OWASP ZAP
    → Proxy web pour l'analyse, interception et modification des requêtes HTTP/S.
  • Sqlmap
    → Outil d'automatisation des tests d'injection SQL.
  • Console developper
    → Permet d'analyser le comportement des sites web et leur sécurité.
  • jwt.io
    → Vérification et décodage de JSON Web Tokens.
  • Gitdumper
    → Récupération d'un projet à partir d'un dossier .git exposé.
  • Ghidra
    → Framework de rétro-ingénierie supportant x86, ARM, MIPS...
  • GDB
    → Débogueur GNU pour l'analyse et l'exploitation de binaires.
  • Les commandes classiques (files, strings, readelf...)
    → Extraction d'informations des binaires.
  • Radar2
    → Outil de reverse engineering en ligne de commande.
  • Cyberchef
    → Outil polyvalent pour transformer, encoder et décoder des données.
  • Hashcat, John
    → Outils de cassage de mots de passe et de hash.
  • OpenSSL
    → Bibliothèque pour manipuler des certificats SSL et chiffrer/déchiffrer des données.
  • Pycryptodome
    → Librairie Python pour le chiffrement et la cryptographie.
  • CrackStation
    → Service en ligne permettant de casser des hash courants.
  • FactorDB
    → Base de données en ligne listant la factorisation de nombres premiers.
  • Wireshark
    → Analyseur de paquets réseau.
  • Dig / whois
